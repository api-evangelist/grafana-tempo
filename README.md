# Grafana Tempo

Grafana Tempo is an open-source, high-scale distributed tracing backend that requires only object storage to operate, making it cost-effective and easy to run. It integrates deeply with Grafana, Prometheus, and Loki for seamless observability across metrics, logs, and traces.

**APIs.json:** [apis.yml](https://raw.githubusercontent.com/api-evangelist/grafana-tempo/refs/heads/main/apis.yml)

## APIs

- **Grafana Tempo HTTP API** — Query traces, search with TraceQL, retrieve tag names and values, and compute metrics summaries from distributed trace data.
  - [OpenAPI](openapi/tempo-http-api.yml)
  - [Documentation](https://grafana.com/docs/tempo/latest/api_docs/)
  - [JSONSchema](json-schema/trace.yml)
